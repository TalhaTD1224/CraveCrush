CraveCrush is an Android-based food ordering application that allows users to browse menus, add items to a cart, and place orders. The app also includes an admin panel for managing menu items and processing orders.

## Features
- User authentication
- Food menu browsing
- Cart & checkout
- Order management
- Admin dashboard

## Tech Stack
- Android (Java/Kotlin)
- Firebase Authentication
- Firebase Firestore
- Material Design
